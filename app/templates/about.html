About EventHub

EventHub is a modern event management platform that makes it easy to discover, create, and manage events. Whether you're organizing a local meetup, a conference, or an online workshop, EventHub provides the tools you need.

Features

  • AI-powered event creation assistant
  • Support for both online and in-person events
  • Chronological event listings
  • User authentication and event management
  • Tag-based event categorization

Technology

EventHub is built with modern web technologies:

  • FastAPI for the backend
  • MongoDB for data storage
  • LangGraph and Ollama for AI-powered event creation
  • HTMX for dynamic interactions
  • Tailwind CSS for styling
